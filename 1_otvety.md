# Обработка изображений — MATLAB

## ЗАДАЧА 1: Геометрические преобразования

### Теоретические сведения

Геометрические преобразования — это операции, изменяющие пространственное расположение пикселей изображения без изменения их интенсивности. Они широко применяются в компьютерном зрении, медицинской визуализации, картографии и обработке спутниковых снимков.

---

### 1. imshow — Отображение изображения

**Назначение:** Выводит изображение в окне MATLAB (Figure). Является основной функцией визуализации в Image Processing Toolbox.

**Синтаксис:**
```matlab
imshow(I)           % Отображение матрицы изображения
imshow('file.jpg')  % Отображение из файла
imshow(I, [])       % Автоматическая настройка контраста
```

**Для каких целей:**
- Визуальный контроль результатов обработки
- Сравнение оригинала и обработанного изображения
- Отладка алгоритмов обработки изображений
- Отображение промежуточных результатов в пайплайне обработки

**Практический код:**
```matlab
% Загрузка изображения
img = imread('peppers.png');

% Отображение
figure;
imshow(img);
title('Исходное изображение');

% Отображение в оттенках серого с автоконтрастом
gray_img = rgb2gray(img);
figure;
imshow(gray_img, []);
title('Серое изображение с автоконтрастом');
```

---

### 2. imcrop — Обрезка изображения

**Назначение:** Извлекает прямоугольную область из изображения. Позволяет выделить область интереса (ROI — Region of Interest).

**Синтаксис:**
```matlab
I2 = imcrop(I, [xmin ymin width height])  % Программная обрезка
I2 = imcrop(I)                             % Интерактивная обрезка (мышью)
```

**Для каких целей:**
- Выделение области интереса (ROI) для дальнейшего анализа
- Удаление ненужных краёв изображения
- Извлечение объектов из сцены
- Подготовка данных для машинного обучения (вырезка лиц, объектов)

**Практический код:**
```matlab
img = imread('peppers.png');

% Обрезка: начиная с точки (50,50), размер 200x150
cropped = imcrop(img, [50 50 200 150]);

figure;
subplot(1,2,1); imshow(img); title('Оригинал');
subplot(1,2,2); imshow(cropped); title('Обрезанное изображение');
```

---

### 3. imresize — Изменение размера

**Назначение:** Изменяет размер изображения путём интерполяции пикселей. Поддерживает увеличение и уменьшение.

**Синтаксис:**
```matlab
I2 = imresize(I, scale)           % Масштабирование (0.5 = уменьшить вдвое)
I2 = imresize(I, [rows cols])     % Указать точный размер
I2 = imresize(I, scale, 'bicubic') % С указанием метода интерполяции
```

**Методы интерполяции:**
- `nearest` — ближайший сосед (быстрый, но грубый)
- `bilinear` — билинейная (баланс скорости и качества)
- `bicubic` — бикубическая (наилучшее качество, по умолчанию)

**Для каких целей:**
- Уменьшение изображений для ускорения обработки
- Увеличение мелких деталей для анализа
- Стандартизация размеров для нейронных сетей
- Создание миниатюр (thumbnail) для предпросмотра

**Практический код:**
```matlab
img = imread('peppers.png');

% Уменьшение в 2 раза
small = imresize(img, 0.5);

% Увеличение до конкретного размера
big = imresize(img, [600 800]);

figure;
subplot(1,3,1); imshow(img); title(sprintf('Оригинал %dx%d', size(img,1), size(img,2)));
subplot(1,3,2); imshow(small); title(sprintf('Уменьшенное %dx%d', size(small,1), size(small,2)));
subplot(1,3,3); imshow(big); title(sprintf('Увеличенное %dx%d', size(big,1), size(big,2)));
```

---

### 4. imrotate — Поворот изображения

**Назначение:** Поворачивает изображение на заданный угол (в градусах) вокруг центра.

**Синтаксис:**
```matlab
I2 = imrotate(I, angle)            % Поворот (+ против часовой, - по часовой)
I2 = imrotate(I, angle, 'bilinear') % С методом интерполяции
I2 = imrotate(I, angle, 'crop')    % С обрезкой до исходного размера
```

**Для каких целей:**
- Коррекция наклона сканированных документов
- Выравнивание спутниковых снимков
- Аугментация данных для обучения нейронных сетей
- Поворот медицинских снимков для удобства анализа

**Практический код:**
```matlab
img = imread('peppers.png');

% Поворот на 45 градусов
rotated = imrotate(img, 45);

% Поворот на 90 градусов с обрезкой
rotated_crop = imrotate(img, 90, 'bilinear', 'crop');

figure;
subplot(1,3,1); imshow(img); title('Оригинал');
subplot(1,3,2); imshow(rotated); title('Поворот 45°');
subplot(1,3,3); imshow(rotated_crop); title('Поворот 90° (crop)');
```

## ЗАДАЧА 2: Работа с элементами изображения

### Теоретические сведения

Помимо геометрических преобразований, важной частью обработки изображений является анализ статистических характеристик: распределение яркости, значения отдельных пикселей, средняя яркость и степень сходства между изображениями.

---

### 1. imhist — Построение гистограммы

**Назначение:** Строит гистограмму распределения интенсивностей пикселей изображения. Гистограмма показывает, сколько пикселей имеют каждое значение яркости (0–255 для 8-битного изображения).

**Синтаксис:**
```matlab
imhist(I)           % Построить и отобразить гистограмму
[counts, x] = imhist(I)    % Получить данные гистограммы
imhist(I, n)        % Гистограмма с n бинами
```

**Для каких целей:**
- Оценка контраста и экспозиции изображения
- Определение порога бинаризации (метод Оцу)
- Диагностика пересвета или недосвета
- Сравнение изображений по распределению яркости
- Основа для эквализации гистограммы (histeq)

**Практический код:**
```matlab
img = imread('peppers.png');
gray = rgb2gray(img);

figure;
subplot(1,2,1); imshow(gray); title('Серое изображение');
subplot(1,2,2); imhist(gray); title('Гистограмма яркости');

% Анализ: если пики слева — тёмное изображение, справа — светлое
% Равномерное распределение — хороший контраст
```

---

### 2. impixel — Получение значения пикселя

**Назначение:** Возвращает значения цветовых каналов (R, G, B) или интенсивность конкретного пикселя по его координатам.

**Синтаксис:**
```matlab
val = impixel(I, col, row)   % Программно по координатам
val = impixel(I)              % Интерактивно (кликом мыши)
```

**Для каких целей:**
- Определение цвета конкретной точки изображения
- Отладка алгоритмов обработки — проверка значений после фильтрации
- Анализ аномальных пикселей (шум, дефекты)
- Калибровка цвета по эталонным точкам
- Извлечение данных для количественного анализа

**Практический код:**
```matlab
img = imread('peppers.png');

% Получить значение пикселя в точке (100, 150)
pixel_val = impixel(img, 150, 100);
fprintf('Пиксель (100,150): R=%d, G=%d, B=%d\n', pixel_val(1), pixel_val(2), pixel_val(3));

% Получить несколько пикселей одновременно
cols = [50, 100, 200];
rows = [50, 100, 200];
vals = impixel(img, cols, rows);
disp('Значения выбранных пикселей:');
disp(vals);
```

---

### 3. mean2 — Среднее значение изображения

**Назначение:** Вычисляет среднее арифметическое всех элементов двумерной матрицы (изображения). Эквивалентно `mean(I(:))`.

**Синтаксис:**
```matlab
m = mean2(I)    % Среднее значение всех пикселей
```

**Для каких целей:**
- Оценка общей яркости изображения (тёмное/светлое)
- Нормализация изображений перед сравнением
- Вычитание среднего для центрирования данных
- Контроль качества — отслеживание изменения средней яркости во времени
- Базовый статистический параметр для алгоритмов обработки

**Практический код:**
```matlab
img = imread('peppers.png');
gray = rgb2gray(img);

% Среднее значение
avg = mean2(gray);
fprintf('Средняя яркость: %.2f (из 255)\n', avg);

% Сравнение: тёмное vs светлое
bright = imadjust(gray, [0 0.5], [0.5 1]);
dark = imadjust(gray, [0.5 1], [0 0.5]);

fprintf('Осветлённое: %.2f\n', mean2(bright));
fprintf('Затемнённое: %.2f\n', mean2(dark));

% Если avg > 128 — изображение скорее светлое
% Если avg < 128 — изображение скорее тёмное
```

### 4. corr2 — Корреляция между изображениями

**Назначение:** Вычисляет коэффициент 2D-корреляции между двумя матрицами одинакового размера. Значение от -1 до +1, где 1 означает полное совпадение.

**Синтаксис:**
```matlab
r = corr2(A, B)    % Коэффициент корреляции (-1 до +1)
```

**Для каких целей:**
- Сравнение двух изображений на степень сходства
- Оценка качества восстановления (фильтрации, сжатия)
- Поиск шаблона (template matching)
- Регистрация изображений — оценка точности совмещения
- Анализ изменений между кадрами видео

**Практический код:**
```matlab
img = imread('peppers.png');
gray = rgb2gray(img);

% Добавить шум
noisy = imnoise(gray, 'gaussian', 0, 0.02);

% Применить фильтр
filtered = medfilt2(noisy);

% Корреляция
r1 = corr2(double(gray), double(noisy));
r2 = corr2(double(gray), double(filtered));

fprintf('Корреляция оригинал-шумное:     %.4f\n', r1);
fprintf('Корреляция оригинал-фильтрованное: %.4f\n', r2);

figure;
subplot(1,3,1); imshow(gray); title('Оригинал');
subplot(1,3,2); imshow(noisy); title(sprintf('С шумом (r=%.4f)', r1));
subplot(1,3,3); imshow(filtered); title(sprintf('Фильтрованное (r=%.4f)', r2));

% r2 > r1 означает, что фильтрация улучшила качество
```
